===============
run-parts
===============

- http://manned.org/run-parts/29d3a6d8

.. contents::
    :local:

ディレクトリ以下のスクリプトを全て実行
=============================================

- 実行はアルファベット順(ASCIIコード, 数字、英大文字、英小文字の順)
- さらにサブディレクトリは無視される

オプション
========================================


-a 各スクリプトに共通の引数を渡す
------------------------------------------------------------

--list  : スクリプトの一覧(実行無し)
------------------------------------------------------------

- 実行しません。一覧のみです。
- --test と同時に使えません。

--regex=RE : 正規表現でファイル名を評価する
------------------------------------------------------------

.. code-block:: bash

    $ run-parts --list --regex '^p.*d$' /etc


制限
=====


Debian
---------

- ファイル名で使える文字 : [0-9A-Za-z-_]
- "."は使えない( .sh とかのファイルは無視 )
- 正規表現オプション使えば回避(だと思う)

CentOS
---------

- 末尾が"~"か","以外のファイルは使える
