3.  Cryptographic Algorithms for JWS
============================================================

JWS uses cryptographic algorithms to digitally sign 
or create a Message Authentication Codes (:term:`MAC`) 
of the contents of the :term:`JWS Header` and the :term:`JWS Payload`.
