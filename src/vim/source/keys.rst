=======
キー
=======


キー
------------

.. glossary::
    :sorted:

    ^
        行頭へ

    $
        行末へ

    %
        対応する括弧へ移動

    ~
        カーソル位置の文字の大文字/小文字を切換える

    b
        <ctrl> b
           PageUp 

    d
        dd
            １行削除

        D
            カーソル位置から行末まで削除

    f    
        <ctrl> f 
            PageDown         

    g
        gg
            ファイルの先頭へ

        G
            ファイルの末尾へ 

    h
        h
            ← (カーソル移動)

        H
            画面上の１行目。 2Hで２行目、{n}Hで{n}行目。

    j   

        j
            ↓ (カーソル移動)

        J
            次の行と連結
        
    l
        l
            → (カーソル移動)

        L
            画面上の最下行へ。2Lで下から２行目、{n}Lで下から{n}行目。        


    M
        画面中央行へ

    x
        １文字削除

    z
        z <enter>
            現在のカーソル位置が画面のトップになるようにスクロール

        zz
            現在のカーソル位置が画面の中央になるようにスクロール
