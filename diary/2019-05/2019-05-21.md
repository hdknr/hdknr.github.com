## 焼肉のたれ

- [「焼肉のたれ」は自家製がオススメ！ 家にある材料で作れる、簡単＆ヘルシーな「自家製焼肉だれ」レシピ - dressing（ドレッシング）](https://www.gnavi.co.jp/dressing/article/21051/)

この量だとちょっとおおいかなともおもうが、瓶で保存すればいいか。

沸騰させてから煮詰める。水分は適当にのこす。

|        |           |
| ------ | --------- |
| 醤油    | 200ml     |
| 根昆布  |  50ml      |
| みりん  |  50ml      |
| さけ    |  50ml     |
| はちみつ |  大さじ1    |
| さとう  |  大さじ1    |

上記と以下をバイタミックスで適当に混ぜる。

|        |           |
| ------ | --------- |
| ニンニク | 1片       |
| ショウガ | 1片       |
| リンゴ  | 1/4個      |
| タマネギ | 1/4個     |
| ごま    | 10g       |

とりだして、ごま油を適当に混ぜる。
