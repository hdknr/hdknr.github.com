SQLite3 backend for django.

Works with either the pysqlite2 module or the sqlite3 module in the standard library.

class django.db.backends.sqlite3.base.SQLiteCursorWrapper[source]¶

Django uses “format” style placeholders, but pysqlite2 uses “qmark” style. This fixes it – but note that if you want to use a literal “%s” in a query, you’ll need to use “%%s”.

django.db.backends.sqlite3.base.decoder(conv_func)[source]¶

The Python sqlite3 interface returns always byte strings. This function converts the received value to a regular string before passing it to the receiver function.
