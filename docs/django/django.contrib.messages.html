django.contrib.messages.api.get_level(request)[source]¶

Returns the minimum level of messages to be recorded.

The default level is the MESSAGE_LEVEL setting. If this is not found, the INFO level is used.

django.contrib.messages.api.set_level(request, level)[source]¶

Sets the minimum level of messages to be recorded, returning True if the level was recorded successfully.

If set to None, the default level will be used (see the get_level method).
