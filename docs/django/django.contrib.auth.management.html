django.contrib.auth.management.get_default_username(check_db=True)[source]¶

Try to determine the current system user’s username to use as a default.

Parameters:check_db – If True, requires that the username does not match an existing auth.User (otherwise returns an empty string).
Returns:The username, or an empty string if no username can be determined.
django.contrib.auth.management.get_system_username()[source]¶

Try to determine the current system user’s username.

Returns:The username as a unicode string, or an empty string if the username could not be determined.
