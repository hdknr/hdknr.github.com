django.contrib.gis.db.backends.oracle.models¶

The GeometryColumns and SpatialRefSys models for the Oracle spatial backend.

It should be noted that Oracle Spatial does not have database tables named according to the OGC standard, so the closest analogs are used. For example, the USER_SDO_GEOM_METADATA is used for the GeometryColumns model and the SDO_COORD_REF_SYS is used for the SpatialRefSys model.
