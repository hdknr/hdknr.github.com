django.core.handlers.base¶

django.core.handlers.base.get_script_name(environ)[source]¶

Returns the equivalent of the HTTP request’s SCRIPT_NAME environment variable. If Apache mod_rewrite has been used, returns what would have been the script name prior to any rewriting (so it’s the script name as seen from the client’s perspective), unless DJANGO_USE_POST_REWRITE is set (to anything).
