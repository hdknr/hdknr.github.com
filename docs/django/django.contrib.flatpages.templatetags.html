django.contrib.flatpages.templatetags.flatpages.get_flatpages(parser, token)[source]¶

Retrieves all flatpage objects available for the current site and visible to the specific user (or visible to all users if no user is specified). Populates the template context with them in a variable whose name is defined by the as clause.

An optional for clause can be used to control the user whose permissions are to be used in determining which flatpages are visible.

An optional argument, starts_with, can be applied to limit the returned flatpages to those beginning with a particular base URL. This argument can be passed as a variable or a string, as it resolves from the template context.

Syntax:

{% get_flatpages ['url_starts_with'] [for user] as context_name %}

Example usage:

{% get_flatpages as flatpages %}
{% get_flatpages for someuser as flatpages %}
{% get_flatpages '/about/' as about_pages %}
{% get_flatpages prefix as about_pages %}
{% get_flatpages '/about/' for someuser as about_pages %}
