django.db.backends.postgresql.version¶

Extracts the version of the PostgreSQL server.

django.db.backends.postgresql.version.get_version(cursor)[source]¶

Returns a tuple representing the major, minor and revision number of the server. For example, (7, 4, 1) or (8, 3, 4). The revision number will be None in the case of initial releases (e.g., ‘PostgreSQL 8.3’) or in the case of beta and prereleases (‘PostgreSQL 8.4beta1’).
