class django.contrib.gis.tests.geo3d.tests.Geo3DTest(methodName='runTest')[source]¶

Only a subset of the PostGIS routines are 3D-enabled, and this TestCase tries to test the features that can handle 3D and that are also available within GeoDjango. For more information, see the PostGIS docs on the routines that support 3D:

http://postgis.refractions.net/documentation/manual-1.4/ch08.html#PostGIS_3D_Functions
