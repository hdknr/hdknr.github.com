class django.contrib.localflavor.hk.forms.HKPhoneNumberField(*args, **kwargs)[source]¶

Validate Hong Kong phone number. The input format can be either one of the followings: ‘XXXX-XXXX’, ‘852-XXXX-XXXX’, ‘(+852) XXXX-XXXX’, ‘XXXX XXXX’, or ‘XXXXXXXX’. The output format is ‘XXXX-XXXX’.

Note: The phone number shall not start with 999, 992, or 112.
And, it should start with either 2, 3, 5, 6, 8, or 9.

Ref - http://en.wikipedia.org/wiki/Telephone_numbers_in_Hong_Kong
