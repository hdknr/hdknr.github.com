Celery本体¶

rogloateでcopytruncateを使う単純な例¶

/var/log/celery/*.log {
        weekly
        missingok
        rotate 52
        compress
        delaycompress
        notifempty
        copytruncate
}

トランザクションが多いとログがロストするでしょう。
