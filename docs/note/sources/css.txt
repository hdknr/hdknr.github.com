行の高さ(line-height)
------------------------------------

::

    line-height : 150%;         // 文字高さ1.5倍
    line-height : 1.5 ;         // 文字高さ1.5倍
    line-height : 1.5em ;       // 文字高さ1.5倍
    

幅(width)
----------

高さ(height)
----------------

rows属性をオーバーライドします。
