Install¶

Debian

$ sudo aptitude install graphviz graphviz-dev ibgraphviz-dev libgraphviz4 -y

Mac Homebrew:

$ brew update
$ brew install gts
$ brew install graphviz

Python

$ pip install pygraphviz --upgrade
