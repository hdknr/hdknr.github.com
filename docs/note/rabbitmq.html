インストール¶

Debian(Wheezy)¶

apt-line:

$ vi /etc/apt/sources.list.d/rabbitmq.list 

deb http://www.rabbitmq.com/debian/ testing main

パブリックキー:

$ wget http://www.rabbitmq.com/rabbitmq-signing-key-public.asc -O /tmp/rabbitmq.key
$ sudo apt-key add /tmp/rabbitmq.key

インストール:

$ sudo aptitude update
$ sudo apt-get install rabbitmq-server

管理ツール:

$ sudo rabbitmq-plugins enable rabbitmq_management
$ sudo /etc/init.d/rabbitmq-server restart

TCPポート:

server$ sudo lsof -c beam  | grep LISTEN

管理¶

仮想ホスト¶

一覧:

(slu)hdknr@wzy:~$ sudo rabbitmqctl list_vhosts

Listing vhosts ...
paloma
/
...done.

追加

(slu)hdknr@wzy:~$ sudo rabbitmqctl add_vhost /sample
Creating vhost "/sample" ...
...done.

ユーザー¶

追加 :

(slu)hdknr@wzy:~$ sudo rabbitmqctl add_user user1 hogehoge1

Creating user "user1" ...
...done.

一覧:

(slu)hdknr@wzy:~$ sudo rabbitmqctl list_users

Listing users ...
paloma  [administrator]
soup    [administrator]
user1   []
guest   [administrator]
...done.

タグ追加 :

(slu)hdknr@wzy:~$ sudo rabbitmqctl set_user_tags user1 administrator

Setting tags for user "user1" to [administrator] ...
...done.

パーミッション¶

追加:

(slu)hdknr@wzy:~$ sudo rabbitmqctl set_permissions -p /sample user1  ".*" ".*" ".*"

Setting permissions for user "user1" in vhost "/sample" ...
...done.

一覧:

(slu)hdknr@wzy:~$ sudo rabbitmqctl list_permissions

Listing permissions in vhost "/" ...
guest   .*      .*      .*
soup    .*      .*      .*
...done.

ユーザーのパーミッション:

(slu)hdknr@wzy:~$ sudo rabbitmqctl list_user_permissions user1

Listing permissions for user "user1" ...
/sample .*      .*      .*
...done.
