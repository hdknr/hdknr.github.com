Git/Mercurial¶

タスク Git Mercural
最初の取得 git clonse {{url}} hg clone {{url}}
最新の取得(のみ)(a) git fetch hg pull
取得したもので更新(b) git merge origin/master hg update [1]
最新を取得して更新(a+b) git pull hg pull -u
[1]hg up, hg checkout, hg co 等でも良い
