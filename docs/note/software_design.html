Flyweight¶

  • 「フライ級」 ( ヘビー、ミドル、ライト 、フライ )
  • 同じインスタンスを共有することで、無駄なインスタンスを生成しないようにして、 プログラム全体を軽くすることを目的としたパターン
  • 共有できるものは共有するという考え方により、 メモリの消費量を節約
  • http://ja.wikipedia.org/wiki/Flyweight_%E3%83%91%E3%82%BF%E3%83%BC%E3%83%B3
