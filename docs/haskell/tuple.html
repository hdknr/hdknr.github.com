タプルとは？¶

  • 複数の異なる型の要素を格納して１つの値にする
  • タプルはサイズが固定 ( リストは拡張できる )

Python¶

>>> (1,'A',"Haskell")
(1, 'A', 'Haskell')

>>> (1,2,3)[0]
1


>>> (1,2,3)[100]
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range

>>> (1,2,3)[-1]
3
>>> (1,2,3)[-2]
2
>>> (1,2,3)[-3]
1
>>> (1,2,3)[-4]
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
