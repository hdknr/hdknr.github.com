なぜブロックするか？¶

  • 検索レーティング で検索サイトから高評価を受けるため
  • 最近のクローラの性能が高いので、根こそぎページを持って行ってしまい、評価を下げる事になってしまいがち
  • 不要なURL/ページをブロックすることは重要

不要なページ¶

  • 完成前のスタブページ / プレースホルダページ
  • 広告しかないページ
  • 他サイトからの引用しかないページ
  • 内容が空か、あるは内容がほとんどないページ
  • ブログなどの日付によるアーカイブ
  • タグ一覧など

制御対象¶

  • クローラー
  • インデクサー

制御方法¶

  • robots.txt : ファイル、種別、ディレクトリなどクロールそのものを拒否
  • meta Robots / X-Robots-Tag : 「クロールは許可するが検索結果には表示しない」
  • Microformats <a rel=”nofollow” ... /> : 「このリンク先には リンクポピュラリティ 評価を継承させない」/ 「リンク先は信頼できない」
