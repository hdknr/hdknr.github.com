表(table)¶

以下のように .. table:: ディレクティブ を使った表に対してもクロスリファレンスを定義することができる。:

.. _xref.table.1:

.. table:: Table 1: JSON Web Signature Reserved Algorithm Values

 +------------------------------+-----------------------------------------------------------+
 | Alg Parameter Value          | Algorithm                                                 |
 +==============================+===========================================================+
 | HS256                        | HMAC using SHA-256 hash algorithm                         |
 +------------------------------+-----------------------------------------------------------+
 | ......                       | ....................................                      |
 +------------------------------+-----------------------------------------------------------+

以下のようにクロスリファレンス書式を使って任意の文中にリンクを置く事で 表1 にジャンプできます。

:ref:`表1 <xref.table.1>`

表１:

Table 1: JSON Web Signature Reserved Algorithm Values
Alg Parameter Value Algorithm
HS256 HMAC using SHA-256 hash algorithm
HS384 HMAC using SHA-384 hash algorithm
HS512 HMAC using SHA-512 hash algorithm
RS256 RSA using SHA-256 hash algorithm
RS384 RSA using SHA-384 hash algorithm
RS512 RSA using SHA-512 hash algorithm
ES256 ECDSA using P-256 curve and SHA-256 hash algorithm
ES384 ECDSA using P-384 curve and SHA-384 hash algorithm
ES512 ECDSA using P-521 curve and SHA-512 hash algorithm
