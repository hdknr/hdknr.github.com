OpenID Connect¶

Based on “Welcome to OpenID Connect ” provided by OpenID Foundation ..

Current Version¶

Basic Client Profile
Basic
OpenID Connect Basic Client Profile 1.0 - Lightweight simple self-contained specification for a web-based Relying Party using the OAuth code flow.
Implicit Client Profile
Implicit
OpenID Connect Implicit Client Profile 1.0 - Lightweight simple self-contained specification for a web-based Relying Party using the OAuth implicit flow.
Discovery
OpenID Connect Discovery 1.0 – (Optional) Defines how user and provider endpoints are dynamically discovered.
Dynamic Registration
Registration
OpenID Connect Dynamic Client Registration 1.0 – (Optional) Defines how clients dynamically register with OpenID Providers.
Standard
OpenID Connect Standard 1.0 – Full HTTP binding specification, for both clients and OpenID Providers; references Messages.
Messages
OpenID Connect Messages 1.0 – Lists all the messages that are used in OpenID Connect. You can use this to create additional bindings for Connect, such as an OpenID Connect binding for XMPP.
Session Management
Session
OpenID Connect Session Management 1.0 – (Optional) Defines how to manage OpenID Connect sessions.
OAuth 2.0 Multiple Response Types
OAuth 2.0 Multiple Response Type Encoding Practices – Registration document for several specific new response types, in accordance with the stipulations of the OAuth Parameters Registry.
