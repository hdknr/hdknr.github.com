Key Material

  • 「キーとなる素材」

    • 共有キー
    • 初期化ベクター (IV)
    • 別のキーを作る為のマスターキー
Secret Key Meterial
  • 秘密の Key Material
Non-Secret Key Material

  • 秘密では無い Key Material

    • 秘密にしない初期化ベクター (IV )
KDF
Key Derivation Functions
  • RFC 2898
  • 共有キーから Secret Key Material を「派生させる」ために使う関数 ( nist-sp-600-56a.5.8)
PBKDF
Password based Key Derivation functions
  • Appendix B RFC 3962
  • http://pypi.python.org/pypi/PBKDF
