1. IntroductionÂ¶

This document defines the link relation type [RFC5988] registrations for the OAuth 2 [RFC6749] authentication framework and for OAuth 1.0a [RFC5849].

These link relation types are used during the discovery process by clients needing to discover the entry points for OAuth authorization, token, and initiation for a service or site.

(draft 07, http://tools.ietf.org/html/draft-wmills-oauth-lrdd-07#section-1 )
